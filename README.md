# FLT — Telegram Mini App «Мой Садик»

> **Это README для разработчиков.** Игра живёт на GitHub Pages: https://eg0r112.github.io/FLT/

## Быстрый деплой

### Amvera (бэкенд + бот)
Переменные: `BOT_TOKEN`, `BOT_USERNAME`, `WEBAPP_URL=https://eg0r112.github.io/FLT`, `CORS_ORIGINS=https://eg0r112.github.io`, `MODE=test`, `AMVERA=1`

### GitHub Pages (фронт)
Settings → Pages → Source: **main** → папка **/docs**

Или: Source → **GitHub Actions** + переменная `API_URL` = URL Amvera.
